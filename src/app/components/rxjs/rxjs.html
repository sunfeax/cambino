What is RxJS?

RxJS (Reactive Extensions for JavaScript) is a library for composing asynchronous and event-based programs using observable sequences.

  • Observables

    Streams of data that can be transformed and combined

  • Operators

    Functions that transform, filter, or combine observables

  • Subscriptions

    Mechanism to execute Observable and receive notifications
